# Scrcpy-Integration-IME
Type using PC keyboard via Scrcpy



If you’ve used Scrcpy and tried typing with your PC keyboard, you may have noticed that English works fine—but Hebrew (and likely other non-Latin languages) does not.

This app solves that problem. It captures your keyboard input as KeyEvent (similar to a physical keyboard) and converts it into proper text input.

Supported languages (currently):
  1. English
  2. Hebrew

You can switch between languages using AltGr on your PC keyboard.

> Note: The app must be set as the active IME (keyboard) on the device.
